# P115StrgmSub - 115网盘订阅追更魔改版

> **MoviePilot 第三方插件**，基于 [mrtian2016/MoviePilot-Plugins](https://github.com/mrtian2016/MoviePilot-Plugins) 的 `P115StrgmSub` 深度定制增强。

**维护：** [jyh](https://github.com/jinyuhao-886)

| 特性 | 官方版 | 魔改版 |

| **规则组管理** | ❌ 无 | ✅ **自动注入 4 套自定义规则**（VIVID/10bit/画质优先）+ 热重载 |

| **网盘洗版** | ❌ 无 | ✅ 评分对比 → 自动转存高分 → 联动删除低分 → 飞书通知 |
| **PT 洗版** | ❌ 无 | ✅ 下载前评分拦截 + 入库清理 + 事件通知 |
| **重命名模板注入** | ❌ 无 | ✅ 统一命名规则，保存即生效 |
| **屏蔽时段调度** | ❌ 无 | ✅ 屏蔽态仅 115 / 开放态恢复全部站点 |
| **兜底自愈** | ❌ 无 | ✅ 恢复初始化配置、修复 _is_rejected 等常见问题 |

## 📦 安装

### 第 1 步：添加自定义插件源

在 MP（MoviePilot）的 **插件源设置** 中添加：

```
https://github.com/jinyuhao-886/MoviePilot-Plugins
```

### 第 2 步：安装插件

重启 MP → 进入插件市场 → 搜索 **115网盘订阅追更** → 安装

> **最低需求：** MP v2.13.3+、115 网盘会员、CloudDrive2

## ⚙️ 核心配置

| 字段 | 说明 |
|------|------|
| `cookies` | 115 网盘登录 Cookie（扫码获取） |
| `save_path` | 电视剧转存目录（如 `/我的接收/MoviePilot/TV`） |
| `movie_save_path` | 电影转存目录 |
| `subscribe_auto_fill` | 开启后新增订阅自动填充规则组 |
| `subscribe_auto_fill_rules` | 二级分类 → 规则组映射（VSelect 多选，已预置常用映射） |
| `block_system_subscribe` | 开启后屏蔽时段仅走 115，不搜公开站 |

### 规则组预设

| 规则组名称 | 说明 |
|------------|------|
| 电视剧非杜比画质优先 | 排除杜比、优先 4K/HDR/高码率 |
| 电视剧杜比画质优先 | 含杜比、优先 4K/高码率 |
| 电影非杜比画质优先 | 排除杜比、优先 4K/HDR/VIVID/高码率 |
| 电影含杜比画质优先 | 含杜比、优先 4K/高码率 |

## 🔄 洗版模式

插件支持**自动洗版升级**功能：

1. **自动扫描**：按配置周期扫描本地 strm 文件评分
2. **评分对比**：对每个剧集/电影，与 115 网盘上同作品对比综合评分
3. **自动转存**：新评分 > 旧评分 + 阈值，自动转存替换
4. **联动删除**：转存成功后自动清理旧 strm + 旧 115 文件（进回收站）
5. **飞书通知**：洗版完成推送通知，标明替换原因

综合评分公式（可切换）：
- **方案一：** 纯体积评分
- **方案二：** 体积×0.75 + 规则因子×0.25

## ❓ 常见问题

**Q：安装后插件显示「未适配当前版本」？**
A：检查 MP 版本是否 ≥ v2.13.3，或尝试修改 `package.v2.json` 中的 `mp_version` 范围。

**Q：规则组不生效？**
A：检查「订阅规则填充」是否开启，以及二级分类是否匹配你选的映射。

**Q：洗版不执行？**
A：确认已开启洗版开关，且 cron 时间在允许窗口内。
